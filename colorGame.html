How to Play

The RGB Color Model

The goal of the game is to guess the color of the button represented by the RGB value shown on the screen. Play in easy mode to get three color options to choose from, play in hard mode to get 6 options to choose from.

The RGB color model helps us represent all possible colors as a set of 3 numbers. Any color imaginable can be represented as a set of three values - (R, G, B). Where R is the amount of "red" in the color, G is the amount of "green" in the color, and B is the amount of "blue" in the color. R, G, and B can only take values between 0 and 255, however.

All the visible colors are made of some fraction of these RGB values. White, for example, is (255, 255, 255), all R, G, and B values are maximum. Black is just (0, 0, 0). Similarly, red is represented as (255, 0, 0), green as (0, 255, 0), and blue as (0, 0, 255).
